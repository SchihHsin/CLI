05  状态模型 — 字段与刷新逻辑
// runtime state { "currentModel": "claude-sonnet-4-5", "highlightedModel": "gpt-5", "modelUsage": { "gpt-5": { "window5h": { "percent": 24, "resetAt": "15:30" }, "window7d": { "percent": 52, "resetAt": "周二 00:00" } } } } // ↑↓ 移动时 highlightedModel = nextModel usage = modelUsage[highlightedModel] risk = Math.max(usage.window5h.percent, usage.window7d.percent) color = risk >= 85 ? "danger" : risk >= 70 ? "warn" : risk >= 50 ? "mid" : "low" renderUsagePanel(usage, color) // ↵ 确认时 previousModel = currentModel currentModel = highlightedModel showSuccessAlert(previousModel, currentModel, modelUsage[currentModel])

07  用量颜色规则
百分比区间状态文案Usage bar 颜色交互处理
0 – 49%健康 / 充足蓝绿只显示用量，不额外提示
50 – 69%中等 / 正常消耗绿黄显示百分比与重置时间，不弹提醒
70 – 84%偏高 / 建议注意黄橙chip 显示「预估偏高」
85 – 100%接近上限橙红插入黄色左线用量提醒；仍允许按 ↵ 切换
08  交互规格说明
场景行为键位
/model 弹出模型选择面板，当前模型置灰并带 ✓，默认高亮第一个推荐候选 ↑↓ ↵ Esc
移动高亮 底部 usage panel 立即刷新当前高亮模型的 5h / 7d 用量百分比与重置时间 ↑↓
搜索筛选 输入关键字后保留 provider 分组；usage panel 跟随筛选后的高亮项 / 或直接输入
接近上限 usage bar 变黄/红，插入黄色左线提醒；仍允许确认切换 ↵ 仍可切换
确认切换 弹窗关闭；绿色左线通知；Sidebar Model 与 status bar 同步新模型 ↵
取消 Esc 退出，不修改当前模型，也不输出消息 Esc
